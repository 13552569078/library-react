## API

### AddButton

| 参数     | 说明          | 类型              | 默认值 |
| -------- | ------------- | ----------------- | ------ |
| text     | 文字说明      | `string`          | 添加   |
| children | -             | `React.ReactNode` | -      |
| showIcon | 是否显示 Icon | `boolean`         | true   |

> Button 组件支持的属性，AddButton 组件也支持。
