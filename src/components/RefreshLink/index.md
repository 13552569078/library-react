## API

### RefreshLink

| 参数      | 说明              | 类型         | 默认值 |
| --------- | ----------------- | ------------ | ------ |
| loading   | 加载状态          | `boolean`    | false  |
| onRefresh | Link onClick 回调 | `() => void` | -      |

> Link 组件支持的属性，RefreshLink 组件也支持。
