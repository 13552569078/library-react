## API

### InputNumberWithLabel

| 参数        | 说明       | 类型      | 默认值    |
| ----------- | ---------- | --------- | --------- |
| label       | 文字说明   | `string`  | -         |
| loading     | 是否加载中 | `boolean` | false     |
| loadingText | 加载文字   | `string`  | 加载中... |

> InputNumber 组件支持的属性，InputNumberWithLabel 组件也支持。
