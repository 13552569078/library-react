# AI Arco Material

AI Arco Material 是 是一个基于 [Arco Design](https://arco.design/react/docs/start) 的 React 组件库，与 Arco Design 设计体系一脉相承。

## ✨ 特性

- 🌈 从业务中来，提炼自业务中各个产品的视觉和交互风格。
- 🎨 与 Arco Design 设计体系一脉相承
- ⚙️ 全链路开发和设计工具体系，与 Umi 框架无缝对接。
- 📦 开箱即用的高质量 React 组件。
- 🛡 使用 TypeScript 开发，提供完整的类型定义文件。
- 🌍 国际化语言支持。
